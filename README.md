PokeEngine is a Python desktop tool that generates a Unity 2022.3+ 2.5D Pokémon-style RPG prototype. It creates project folders, JSON databases, runtime C# systems, editor tools, tests, and a playable prototype scene with movement, camera modes, encounters, catching, battle UI, party leveling, save/load, raids, and starter data.
