# PokeEngine
PokeEngine is a desktop Unity 2022.3.62f3 project generator for Pokémon-inspired RPG prototypes. It creates folders, scripts, JSON data, docs, QA files, and a playable starter scene with runtime systems for overworld, movement, NPCs, battles, raids, saves, fusion, Mega Evolution, and Dimension Split.

PokeEngine is a desktop Unity project generator for building a Pokémon-inspired RPG prototype. It combines a modern dark UI, one-click Unity project generation, patchable generator code, and a full starter architecture for overworld exploration, Pokémon data, battles, raids, saves, events, and tooling.

The app generates a Unity 2022.3.62f3 project with folders, scripts, JSON databases, docs, QA files, project settings, and an auto-created PrototypeRegion scene. The generated scene includes a camera, sun light, runtime bootstrap object, starter terrain chunks, NPC markers, player visual, and prototype systems ready to expand.

Key features include:

Desktop project builder UI
Patchable generator core with automatic backups
Unity project/scaffold generation
Auto-generated prototype scene
Pokémon database, moves, learnsets, encounters, evolutions
Overworld streaming and 2.5D camera foundation
Hybrid movement and NPC systems
Battle, damage, ability, Mega Evolution, Dimension Split, Fusion, and Raid scaffolds
Save/autosave system
Generated docs, QA plan, and editor/tooling structure
